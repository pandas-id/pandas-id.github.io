Belajar Ngoding Di Android

Halo sobat! Mungkin dari Kamu ada yang ingin belajar pemrograman/ngoding tapi mengalami beberapa kendala. Kamu tidak punya laptop/PC tapi sangat ingin untuk belajar pemrograman/coding atau ada juha beberapa dari Kamu yang sekedar ingin merasakan sensasi Ngoding di Android.

Lantas muncul pertanyaan dari Kamu.Apakah bisa Ngoding di Android? Untuk menjawab pertanyaan itu silahkan baca artikel ini sampai selesai.

Pengertian Ngoding

Sebagai orang awam,Kamu mungkin tidak tahu menahu tentang Ngoding.Ngoding atau Coding adalah kegiatan menuliskan script menggunakan Bahasa Pemrograman.

Dalam proses coding Kamu harus mengikuti atauran yang berlaku.Aturan tersebut harus sesuai dengan bahasa pemrograman yang Kamu pakai.

Dengan kata coding adalah memberikan perintah kepada komputer untuk melakukan susuatu.

Coding di Android

Di zaman sekarang ini bukan tidak mungkin untuk coding di Android. Meskipun ada beberapa batasan tapi,minimal Kamu mengerti dasar-dasar pemrograman seperti percabangan,pengkondisian,fungsi,dan lain sebagainya.

Berikut ini adalah tata cara untuk coding di Android

Bahasa Pemrograman

Sebelum mulai coding ada baiknya Kamu telah menentukan bahasa pemrograman yang akan Kamu pakai.Dalam hal ini Kamu bisa tentukan dengan melihat tujuan Kamu kedepannya.

Misalnya,Kamu mau menjadi Web Developer,maka Kamu bisa mulai belajar dari HTML,CSS,lalu terakhir adalah Javascript.Itu untuk body dari websitenya,lalu untuk enginenya,Kamu bisa memilih antara Python,NodeJs,PHP.

Kabanyakan orang akan memilih PHP.Akan tetapi mempelajari ketiganya juga tidak masalah:).

Text Editor

Dalam dunia programming text editor adalah alat yang sangat penting bagi programmer.Sebagai catatan di sini,programmer adalah orang yang melakukan kegiatan ngoding.

Saat ini sudah banyak aplikasi-aplikasi text editor yang tersedia di Play Store dengan fitur yang sudah bisa dibilang cukup untuk memenuhi kebutuhan Kamu.Di bawah ini ada beberapa rekomendasi aplikasi yang bisa Kamu pakai untuk coding di Android.

Acode

Acode adalah editor kode dan IDE web yang ringan namun kuat untuk android. Kamu dapat menggunakan editor ini untuk mengedit HTML, JavaScript, teks.

Acode juga merupakan salah satu text editor andalan Admin untuk ngoding di Android. Di sini Kamu dapat membuat situs web, menjalankan situs web di browser dan melihat kesalahan atau log di konsol. Juga edit segala jenis file sumber seperti, python, CSS, html, java, JavaScript, Dart dll.

CATATAN: Kamu tidak dapat menjalankan program selain HTML, MarkDown dan JavaScript, berarti Kamu tidak dapat menjalankan Python, PHP, Java, C / C ++, dll, tetapi Kamu dapat menggunakan Termux di sepanjang Acode untuk menjalankan program ini.

Fitur

  • Mendukung penyorotan sintaks untuk 100+ bahasa pemrograman
  • Puluhan tema
  • User friendly
  • Pratinjau HTML / Tandai Down dalam Aplikasi
  • Konsol JavaScript interaktif
  • Pintasan keyboard

TrebEdit

TrebEdit adalah editor HTML untuk desain web.

Di TrebEdit Kamu dapat mulai membuat proyek web sekaligus langsung melihat pratinjaunya dengan mudah di browser bawaan aplikasinya.

Fitur

  • Dukungan Keyboard dengan keybindings dikonfigurasi

Admin rasa cuma dua hal itu yang menjadi elemen penting untuk coding di Android.Selanjutnya Kamu bisa langsung coding.

Kesimpulan

Coding di Android memang tak semudah yang dibayangkan.Ada banyak kendala dan batasan-batasan yang berlaku.Contohnya layar yang kecil,ini menyulitkan beberapa orang untuk membaca codingan yang mereke buat.

Jika kamu ingin membuat aplikasi atau game di Android,mungkin akan lebih sulit.Tapi bukan berarti tidak bisa ya:)

Terlepas dari semua kekurangan itu mari kita kuatkan niat dan terus berusaha agar yang kita inginkan tercapai.
