Fungsi Dan Kegunaan Termux

Halo sobat! Tentu dari kalian banyak yang bertanya-tanya mengenai aplikasi yang satu ini. Apa itu termux? Apa fungsi dan kegunaannya? Apakah Termux bisa dipakai untuk melakukan kegiatan Hacking? Mungkin, itu adalah beberapa pertanyaan yang sering kalian ajukan.Tapi tenang saja,pada kesempatan kali ini Admin akan menjawab semua pertanyaan di atas.Jadi,silahkan baca artikel ini samapai selesai:)

Apa itu Termux?

Termux adalah emulator terminal Android dan aplikasi lingkungan Linux yang bekerja langsung tanpa rooting atau pengaturan tertentu.Termux yang juga termasuk lingkungan Linux berbasis Debian yang bisa mendukung perangkat berarsitektur 32-bit dan 64-bit dengan dijalankan pada OS Android versi 5.0 ke atas.

Dengan kata lain,Termux adalah Terminal Emulator yang di jalankan di Android.

Perlu diketahui juga,terminal yang dikatakan di sini bulanlah terminal bus seperti yang kalian pikirikan,hehehe,tapi,Terminal adalah sebuah command prompt dimana kita bisa mengontrol file, membuat folder, ataupun membaca, membuat, merubah file, dan masih banyak lagi fungsi yang lainnya.

Apa fungsi dan kegunaannya?

Mungkin pertanyaan inilah yang paling sering orang awam ajukan.Tapi,tenang saja karna atikel ini juga akan membahas mengenai hal itu.

Berikut beberapa fungsi Termux yang telah Admin rangkum.

1. Mempelajari perintah dasar linux

Kalian ingin mempelajar perintah dasar linux di Android? Mengapa tidak?

Termux pada dasarnya terminal emulator yang digunakan untuk mengakses shell linux di android mu,jadi perintah linux perintah termux juga.

Untuk memulai belajar termux minimal kamu harus tau dan hafal perintah dasar linux,jangan hanya cuma mengetik perintah yang ada di tutorial tanpa tau maksud dan fungsi dari perintah tersebut. (Admin juga pernah begini..hehe)

2. Sebagai Text Editor untuk Ngoding

Yap,sebagai alat buat ngoding.Mungkin ada beberapa dari kalian yang membaca artikel ini tidak tau dengan ngoding.Singkatnya Ngoding adalah kegiatan menulis kode program.

Jadi,Termux menyediakan beberapa package yang berfungsi sebagai text editor.Tentu saja Text Editor yang saya maksud adalah Text Editor berbasis Terminal. Beberapa Text Editor yang dapat kalian gunakan diantaranya adalah:

  • Vim
  • NeoVim
  • Nano
  • Micro
  • DLL.

3. Menjalankan Bahasa Pemrograman

Selain untuk ngoding,di Termux kalian juga bisa langsung menjalankan codingan yang kalian tulis.Beberapa bahasa pemrograman yang dapat kalian jalankan di Termux di antara sebagia berikut:

  • Python
  • Ruby
  • Perl
  • Lua
  • DLL.

4. Digunakan untuk SSH Client

Remote SSH di Android? Mengapa tidak? SSH atau Secure Shell digunakan untuk masuk ke komputer jarak jauh dan menjalankan perintah, transfer file dan untuk melakukan banyak hal lainnya.Hal ini juga dapat kalian lakukan menggunakan Termux.

5. Termux sebagai Web Server

Di Termux kalian juga bisa juga menjalankan web server.Web server di sini adalah perangkat lunak yang berfungsi sebagai penerima permintaan yang dikirimkan melalui browser kemudian memberikan tanggapan permintaan dalam bentuk halaman situs web atau lebih umumnya dalam dokumen HTML.

Ada beberapa web server yang dapat kalian gunakan di termux, di antaranya adalah:

  • Apache
  • PHP
  • NodeJS

Itu tadi beberapa fungsi dan kegunaan dari Aplikasi Termux.Cukup multifungsi bukan?

Apakah Termux bisa digunakan untuk Hacking?

Jawabannya adalah ya.Jika kalian ingin menggunakan Termux untuk kegiatan tersebut,maka kalian dapat menginstal tools seperti Hydra,Sqlmap,Nmap,Metasploit Framework, dan lain sebegainya.

Namun pada artikel ini,Admin tidak akan membahas Bagaimana Cara Hack FB atau bagaimana cara Deface Website.Kenapa? Karna Admin juga tidak tau...hehehe:)

Kesimpulan

Termux adalah sebuah aplikasi yang multifungsi dan bisa digunakan untuk berbagai keperluan.Bukan hanya yang berkaitan dengan Hacking tapi bisa juga digunakan untuk belajar Programming

Itu tadi pembahasan mengenai apa itu Termux dan berbagai kegunaannya.Harapan Admin,semoga artikel ini bisa menambah pengetahuan kita dan menjadikan kita seseorang yang bisa memanfaatkan teknologi dengan sebaik-baiknya.

Akhir kata,salam sejahtera buat kita semua dan sampai jumpa...
