Cara Menyimpanan Video Dari Facebook Ke Galeri

Halo teman-teman semua,pada kesempatan kali ini saya akan membagikan sebuah tutorial untuk mendownload video dari Facebook lalu menyimpannya ke galeri

Sebelum masuk ke tutorialnya,sebaiknya kita menyiapkan aplikasi yang akan kita gunakan terlebih dahulu,aplikasi itu adalah Termux.Kalian bisa mengunduhnya di Play Store atau langsung klikdi sini.

Cara mendownload video dari Facebook

Langkah pertama,silahkan buka Aplikasi Termux lalu masukan perintah di bawah.

pkg install git

Perintah tersebut berfungsi untuk menginstal package git.Package git adalah sebuah Package yang nantinya akan digunakan untuk menginstal/mengclone script dari github.

Langkah selanjutnya adalah menginstal bahasa pemrograman,bahasa pemrograma yang akan Kita install adalah Bahasa Pemrograman Python.

pkg install python

Bahasa Pemrograman Python nantinya akan digunakan untuk menjalankan script yang tentunya dibuat juga dengan Bahasa Pemrograman Python.

Langkah berikutnya adalah menginstal modul,modul yang akan di instal adalah modul requests.

Sekarang semua package dan modul telah kita instal,maka yang selanjutnya akan kita lakukan adalah menginstal/mengclone script dari github.Untuk lebih jelasnya silahkan masukan perintah di bawah.

git clone https://github.com/pandas-id/fb-video-downloader

Jika proses di atas telah selesai maka akan ada folder baru yang bernam fb-video-downloader.Untuk melihatnya silahkan masukan perintah ls lalu tekan Enter.

ls

Masuk ke folder tersebut dengan menggunakan perintah cd.

cd fb-video-downloader

Jalankan program dengan menggunakan perintah di bawah.

python downloader.py

Silahkan cari video yang akan kalian download lalu copy urlnya dan pastekan seperti di bawah,jangan lupa juga masukan tempat di mana video akan disimpan.

Tunggu beberapa saat maka video yang kalian download telah tersimpan di galeri hp kalian.

Jika penjelasan di atas kurang di pahami silahlan nonton videonya di sini.

Sampai di sini saja ya teman-teman semuanya,semoga apa yang saya bagikan bermanfaat bagi kalian semua.
